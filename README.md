# shiroi_frame

#### 介绍
shiroi 自制框架 (第一个脚手架)

#### 软件架构
初始的目录结构如下：
~~~
├─config                所有配置的目录
|   ├─config.php        基础配置文件
|   └─database.php      数据库配置文件
├─index                 应用目录(可自定义|无要求)        
├─shiroi                框架系统目录
|   ├─resource          框架类库目录
|   |  ├─Config.php     模块配置文件
|   |  ├─FileHandle.php 文件操作类文件
|   |  ├─Loader.php     框架注册类文件
|   |  └─Route.php      路由注册类文件
|   ├─base.php          基础定义文件
|   └─run.php           框架入口文件
├─vendor                第三方类库目录（Composer依赖库）
├─composer.json         composer 定义文件
├─README.md             README 文件
├─.gitignore            git忽略文件
~~~
